# Basic Express App Starter
- Built with [express-generator](https://www.npmjs.com/package/express-generator)

## To clone app:
`
git clone https://github.com/ntomkevich/basic_express_app.git
`

## To run app:
#### Install dependencies:

`
npm install
`

#### Start app:

`
npm run dev
`

#### Browse app at:

`
localhost:3000
`
